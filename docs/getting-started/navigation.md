# Navigation basics

> How to use the joystick, buttons, and menus on your SeedSigner device.

SeedSigner uses a simple set of physical controls — a joystick and three buttons. Once you understand how they work, navigating the interface is straightforward.

![Waveshare LCD HAT showing joystick on the left and three buttons (A, B, C) on the right](../images/WaveShare_LCD_Hat.png)

## Controls overview

| Control | Function |
|---------|----------|
| **Joystick (up/down/left/right)** | Move between menu items, scroll through options, and highlight on-screen elements |
| **Joystick (center press)** | Select the currently highlighted item (same as pressing any key) |
| **Key A** | Select / confirm |
| **Key B** | Select / confirm |
| **Key C** | Select / confirm |

> **Tip:** In most menus, Keys A, B, and C all do the same thing — they select or confirm whatever is currently highlighted on screen. In some contexts (such as the word entry keyboard), each key has a distinct function. SeedSigner labels these on screen when they differ.

## Moving through menus

Use the **joystick** to move up and down through menu lists. The currently highlighted item is visually distinct on screen. Once the item you want is highlighted, press **any key** (A, B, C, or joystick center) to select it.

## Going back

Most screens display a **back arrow** in the top-left corner. To go back:

1. Use the joystick to highlight the back arrow.
2. Press any key to return to the previous screen.

This works consistently throughout the entire interface. If you ever navigate somewhere by mistake, just go back.

## Accessing the power menu

In the top-right corner of the main menu you will see a **power icon**. Highlight it with the joystick and press any key to open the power menu, where you can power off or restart the device.

For full details on shutting down, see [Power off and restart](/getting-started/power-off.md).
